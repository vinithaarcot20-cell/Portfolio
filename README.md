# 🌟 Vinitha's Personal Portfolio

A refined, dark-themed personal portfolio website built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no dependencies.

![Portfolio Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

---

## ✨ Features

- **Elegant Dark Theme** — Luxury gold accent palette with noise texture overlay
- **Smooth Animations** — Staggered page-load reveals and scroll-triggered effects
- **Animated Skill Bars** — Bars animate into view as you scroll down
- **Responsive Design** — Fully optimized for mobile, tablet, and desktop
- **Interactive Contact Form** — With visual submission feedback
- **Sticky Navbar** — Frosted glass effect on scroll
- **Zero Dependencies** — Pure HTML, CSS & JS, no libraries required

## 🚀 Getting Started

### View Locally
Just open `index.html` in your browser — no build step needed!

```bash
git clone https://github.com/vinithaarcot20-cell/Portfolio.git
cd Portfolio
open index.html
```

### Deploy with GitHub Pages
1. Go to **Settings → Pages**
2. Set source branch to `main`
3. Click **Save**
4. Your site will be live at:
   ```
   https://vinithaarcot20-cell.github.io/Portfolio/
   ```
